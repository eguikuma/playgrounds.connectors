# my.http-connectors

## 概要

HTTP通信を型安全に抽象化します。

結果をSuccessとFailedとして扱うことができます。

## 機能

### パッケージ構成

| 名前 | 説明 |
|-----------|------|
| @outcomify/requestify | 通信基盤 |
| @outcomify/restify | RESTful API（スタンドアロン） |
| @outcomify/restify-tanstack | RESTful API（TanStack Query） |
| @outcomify/rssify | RSSフィード処理（スタンドアロン） |
| @outcomify/rssify-tanstack | RSSフィード処理（TanStack Query） |
